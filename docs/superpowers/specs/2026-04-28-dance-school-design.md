# 明心慧志舞蹈学校 宣传网站设计规范

## 概述

明心慧志舞蹈学校（中国舞）面向大众的宣传网站。兼顾品牌展示与课程转化，采用新中式水墨留白风格。单页滚动式，响应式适配桌面与移动端。

## 设计方向

新中式·水墨留白风。黑白水墨基底，朱砂红点缀，靛蓝辅助。灵感源于宣纸书画，强调"气韵"与留白，通过克制动画营造行云流水的东方审美体验。

## 字体

- 标题：Noto Serif SC (思源宋体) → fallback: STSong, Songti SC, serif
- 正文：Noto Sans SC (思源黑体) → fallback: PingFang SC, Microsoft YaHei, sans-serif

## 页面结构（单页滚动）

1. **导航栏** — 固定顶部，半透明毛玻璃效果，校名 + 锚点链接
2. **Hero 首屏** — 大幅水墨视觉背景，校名逐字动画，核心标语，向下滚动提示
3. **关于·明心慧志** — 品牌理念段，左侧文字 + 右侧装饰性水墨元素
4. **课程体系** — 卡片网格展示（少儿启蒙/少儿进阶/成人班/艺考辅导），悬停浮起
5. **师资风采** — 横滑或网格展示教师照片 + 简介
6. **学员风采** — 照片画廊，瀑布流或网格
7. **家长心声** — 简短感言卡片（可选模块）
8. **联系我们** — 地址/电话/微信 + 引导试课 CTA
9. **页脚** — 版权信息，简洁收尾

## 动画规范

- 总原则：克制从容，不喧宾夺主
- 时长：0.3s–0.8s，曲线 ease-out
- 滚动淡入：模块进入视口时从下方淡入上浮（Intersection Observer）
- 水墨过渡：模块间用浅色 SVG 笔触分隔
- Hero 标题：逐字渐现
- 导航悬停：底部朱砂红细线延伸
- 按钮悬停：颜色加深，微缩放
- 卡片悬停：上浮 + 阴影增强
- 背景纹理：极淡的随机墨点/墨痕
- Hero 视差：轻微纵向视差

## 技术方案

- 纯静态 HTML/CSS/JS，无框架
- Google Fonts 加载思源宋体 + 思源黑体
- Intersection Observer API 驱动滚动动画
- CSS 变量管理配色
- 响应式：Flexbox/Grid，桌面优先，移动端适配
- 所有视觉素材用 CSS/SVG 占位（无实际图片，留有替换入口）
